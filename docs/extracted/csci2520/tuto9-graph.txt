# Slide 3
Depth First Search (DFS)
DFS is an algorithm for traversing or searching tree or graph data structures.
1. Select a root node
2. Explores as far as possible along each branch before backtracking
Usually use a stack to keep track of node path
Use a bool array to tag visited

# Slide 20
Breadth First Search (BFS)
BFS is an algorithm for traversing or searching tree or graph data structures.
1. Select a root node
2. Explores all nodes at the present depth prior to moving on at the next depth level
Usually use a queue to keep track of children
Use a bool array to tag visited

Prim’s Algorithm (MST)
Prim’s algorithm is a greedy algorithm that’s finds a MST for a weighted undirected graph.
Building this tree one vertex at a time
1. Start from an arbitrary vertex
2. Adding the cheapest possible connection from the tree to another vertex

# Slide 40
Kruskal’s Algorithm (MST)
Find an MST of an undirected weighted graph.
A greedy algorithm
1. Sort the edges.
2. In each step it adds the next lowest-weight edge that will not form a cycle to the minimum spanning forest.

# Slide 47
Dijkstra (Shortest Path)
Finding the shortest paths between nodes in a weighted graph.
While not all vertices are visited
Pick the unvisited vertex 
v
 whose current best cost is minimum.
For any other vertex 
u
 that can be reached cheaper via 
v
Update the current best cost and path of 
u
.
