# Slide 24
Quickselect
 Complexity (Optional)
Intuitively, it is n + n/2 + n/4 + … + 1 = 2n-1 = O(n)
Strictly, 
average case
 time complexity T(n) = O(n) + T(n/2)
T(n) ≤ c × n + T(n/2)
T(n/2) ≤ c × n/2 + T(n/4)
T(n/4) ≤ c × n/4 + T(n/8)
……
T(2) ≤ c × 1 + T(1)
T(1) = c’
So, T(n) ≤ c × (n + n/2 + n/4 + … + 1) + c’ = c × (2n-1) + c’
T(n) = 
O(n)
!
