# 个人网站模板

> 📝 **模板说明**: 这是一个示例项目，展示项目展示页面的功能。使用模板时请删除或替换此文件，添加你自己的项目。

这是一个功能完善的个人网站模板项目，包含以下特性：

## 技术栈

- **框架**: Next.js 15 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS + shadcn/ui
- **内容**: MDX + Contentlayer
- **数学公式**: KaTeX

## 功能特性

- 响应式设计
- 暗色模式支持
- SEO 优化
- 玻璃拟态 UI
- 博客系统
- 项目展示
- 数学笔记（支持 LaTeX）
