---
title: "4.1 Homogeneous systems and null space"
description: "Use homogeneous systems and null spaces to describe all solutions systematically, not just one solution at a time."
---

By the time you can row-reduce a system, the next question is no longer
"Can I solve this example?" but rather "What is the structure of every
solution?" Homogeneous systems are the cleanest place to ask that question, and
the null space is the language that answers it.

## Why homogeneous systems are special

A homogeneous linear system is a system whose constant terms are all `0`. In
matrix form, it looks like

$$
Ax = 0.
$$

This situation is special for one immediate reason: the zero vector always
solves it.

<Definition title="Homogeneous system">
A homogeneous linear system is a linear system of the form

$$
Ax = 0.
$$

Its trivial solution is the zero vector `x = 0`.
</Definition>

The real question is whether there are also nontrivial solutions.

## The null space collects all homogeneous solutions

<Definition title="Null space">
If `A` is a matrix, the null space of `A` is

$$
N(A) = \{x : Ax = 0\}.
$$

So `N(A)` is exactly the solution set of the homogeneous system `Ax = 0`.
</Definition>

This definition turns a list of solutions into a mathematical object. Instead
of saying "here are some vectors that work," you can describe the whole set at
once.

## Row reduction tells you the shape of the null space

To find `N(A)`, you solve `Ax = 0` by reducing the augmented system `[A \mid 0]`.
The pivots tell you which variables are determined; the free variables tell you
how many directions of freedom remain.

<WorkedExample title="Solve a homogeneous system and describe the null space">
Let

$$
A =
\begin{bmatrix}
1 & 2 & -1 \\
2 & 4 & -2
\end{bmatrix}.
$$

To solve `Ax = 0`, row-reduce:

$$
\left[
\begin{array}{ccc|c}
1 & 2 & -1 & 0 \\
2 & 4 & -2 & 0
\end{array}
\right]
\sim
\left[
\begin{array}{ccc|c}
1 & 2 & -1 & 0 \\
0 & 0 & 0 & 0
\end{array}
\right].
$$

So the equation is

$$
x_1 + 2x_2 - x_3 = 0.
$$

Take `x_2 = s` and `x_3 = t` as free variables. Then

$$
x_1 = -2s + t.
$$

Therefore

$$
x =
\begin{bmatrix}
-2s + t \\
s \\
t
\end{bmatrix}
=
s
\begin{bmatrix}
-2 \\ 1 \\ 0
\end{bmatrix}
+
t
\begin{bmatrix}
1 \\ 0 \\ 1
\end{bmatrix}.
$$

So

$$
N(A) =
\operatorname{Span}
\left\{
\begin{bmatrix}
-2 \\ 1 \\ 0
\end{bmatrix},
\begin{bmatrix}
1 \\ 0 \\ 1
\end{bmatrix}
\right\}.
$$
</WorkedExample>

This example shows why null-space descriptions are powerful. They tell you not
only whether solutions exist, but how every solution is built.

## Homogeneous solutions control nonhomogeneous ones

The same idea explains the structure of a system `Ax = b` when it is
consistent.

<TheoremCard title="Every solution is a particular solution plus a null-space vector">
Suppose `x_p` is one particular solution of `Ax = b`.

Then a vector `x` solves `Ax = b` if and only if

$$
x = x_p + v
$$

for some `v \in N(A)`.
</TheoremCard>

This is the key structural theorem behind free-variable formulas.

<CollapsibleProof title="Why the full solution set has the form `x_p + N(A)`">
First suppose `x` is any solution of `Ax = b`. Then

$$
Ax = b
\qquad \text{and} \qquad
Ax_p = b.
$$

Subtracting gives

$$
A(x - x_p) = 0.
$$

So `x - x_p \in N(A)`, which means `x = x_p + v` for some `v \in N(A)`.

Conversely, if `v \in N(A)`, then `Av = 0`. Hence

$$
A(x_p + v) = Ax_p + Av = b + 0 = b.
$$

So every vector of the form `x_p + v` is also a solution.
</CollapsibleProof>

## A nonhomogeneous example

<WorkedExample title="Describe all solutions as a translate of the null space">
Suppose the system `Ax = b` has one particular solution

$$
x_p =
\begin{bmatrix}
3 \\ 0 \\ 1
\end{bmatrix},
$$

and suppose

$$
N(A) =
\operatorname{Span}
\left\{
\begin{bmatrix}
1 \\ -1 \\ 0
\end{bmatrix}
\right\}.
$$

Then every solution has the form

$$
x =
\begin{bmatrix}
3 \\ 0 \\ 1
\end{bmatrix}
+
t
\begin{bmatrix}
1 \\ -1 \\ 0
\end{bmatrix}
=
\begin{bmatrix}
3 + t \\ -t \\ 1
\end{bmatrix},
\qquad t \in R.
$$

The null space gives the direction of freedom; the particular solution tells
you where that family of solutions sits.
</WorkedExample>

## Why free variables force infinitely many homogeneous solutions

The reduced system viewpoint makes one important consequence immediate.

<TheoremCard title="A free variable creates infinitely many solutions">
If the homogeneous system `Ax = 0` has at least one free variable, then it has
infinitely many solutions.
</TheoremCard>

The reason is not mysterious. A free variable may be assigned any real value,
and each different assignment gives a different solution vector unless the
parameter disappears completely.

This also gives a short theorem that is worth stating explicitly: if a
homogeneous system has more variables than pivot equations, then at least one
free variable remains, and the system must therefore have infinitely many
solutions.

<WorkedExample title="One free variable already produces a whole line of solutions">
Suppose row reduction shows that

$$
x_1 - 3x_2 = 0.
$$

If `x_2 = t`, then `x_1 = 3t`, so every solution has the form

$$
\begin{bmatrix}
x_1 \\ x_2
\end{bmatrix}
=
t
\begin{bmatrix}
3 \\ 1
\end{bmatrix},
\qquad t \in R.
$$

Different values of `t` give different vectors, so the homogeneous system has
infinitely many solutions, not just more than one.
</WorkedExample>

<WorkedExample title="A trivial null space can also occur">
Let

$$
A =
\begin{bmatrix}
1 & 0 \\
0 & 1
\end{bmatrix}.
$$

Then `Ax = 0` is simply

$$
x_1 = 0, \qquad x_2 = 0.
$$

So the only solution is the zero vector, and therefore

$$
N(A) = \{0\}.
$$

This is the opposite extreme from the earlier example with free variables.
</WorkedExample>

## The null space is a subspace

This fact is easy to overlook because we first meet the null space as a
solution set. But it is more than a solution set: it is always a subspace of
the domain of `A`.

<TheoremCard title="The null space is closed under linear combinations">
For any matrix `A`, the null space `N(A)` is a subspace. In particular:

- `0 \in N(A)`,
- if `u, v \in N(A)`, then `u + v \in N(A)`,
- if `u \in N(A)` and `c` is a scalar, then `cu \in N(A)`.
</TheoremCard>

<CollapsibleProof title="Why the null space is a subspace">
We already know `0 \in N(A)` because `A0 = 0`.

Now take `u, v \in N(A)`. Then `Au = 0` and `Av = 0`. By linearity,

$$
A(u+v) = Au + Av = 0 + 0 = 0,
$$

so `u+v \in N(A)`.

Likewise, if `u \in N(A)` and `c` is a scalar, then

$$
A(cu) = c(Au) = c \cdot 0 = 0,
$$

so `cu \in N(A)`.
</CollapsibleProof>

This matters later because once you know a solution set is a subspace, you may
look for a basis, count dimensions, and compare it to the pivot structure of
the coefficient matrix.

## Nullity counts how many independent directions remain

The previous discussion explains why the null space is not just a pile of
solutions. It records how many genuinely independent directions of motion are
still left after the pivot equations have imposed all their constraints.

Each free variable contributes one independent parameter. So the dimension of
the null space is exactly the number of free variables in the reduced
homogeneous system.

In rank language, this becomes

$$
\operatorname{nullity}(A) = n - \operatorname{rank}(A),
$$

but even before that theorem is named formally, you should already read nullity
as "the number of independent null-space directions left by the system."

<WorkedExample title="Membership in the null space is a direct test">
Let

$$
A =
\begin{bmatrix}
1 & 1 & 0 \\
0 & 1 & 1
\end{bmatrix},
\qquad
x =
\begin{bmatrix}
1 \\ -1 \\ 1
\end{bmatrix},
\qquad
z =
\begin{bmatrix}
1 \\ 0 \\ 0
\end{bmatrix}.
$$

Then

$$
Ax =
\begin{bmatrix}
0 \\ 0
\end{bmatrix},
\qquad
Az =
\begin{bmatrix}
1 \\ 0
\end{bmatrix}.
$$

So `x \in N(A)` but `z \notin N(A)`. This is the practical meaning of the
definition: to test membership in `N(A)`, compute `Ax` and check whether the
result is exactly the zero vector.
</WorkedExample>

## How to read a basis for the null space from RREF

In practice, the basis vectors of `N(A)` come directly from the free-variable
description of the reduced system.

The workflow is:

1. row-reduce `A`,
2. identify pivot and free variables,
3. set one free variable to `1` and the others to `0`,
4. solve for the pivot variables,
5. repeat once for each free variable.

The vectors produced this way form a basis candidate for the null space because
each one records one independent direction of freedom.

There is also a geometric distinction worth keeping clear:

- a homogeneous solution set is always a subspace, so it passes through the
  origin;
- a nonhomogeneous consistent solution set is usually a translate of that
  subspace by a particular solution.

That difference is exactly why `N(A)` is the structural core of the system,
while `x_p + N(A)` describes the full solution set of `Ax = b`.

## Homogeneous solutions and column dependence

The null-space equation also explains when the columns of a matrix are
dependent.

Let the columns of `A` be `a_1, a_2, \ldots, a_n`. Then

$$
Ax = 0
$$

is the same as

$$
x_1 a_1 + x_2 a_2 + \cdots + x_n a_n = 0.
$$

So a nontrivial solution of the homogeneous system is exactly a nontrivial
linear relation among the columns.

<WorkedExample title="A nontrivial null-space vector gives a dependence relation">
Suppose

$$
x =
\begin{bmatrix}
1 \\ -2 \\ 1
\end{bmatrix}
\in N(A).
$$

Then

$$
Ax = 0
$$

means

$$
1 \cdot a_1 - 2 \cdot a_2 + 1 \cdot a_3 = 0.
$$

This is a nontrivial dependence relation among the columns of `A`.
</WorkedExample>

## What null space says about uniqueness

The structure theorem gives an immediate test.

- If `N(A) = \{0\}`, then a consistent system `Ax = b` has exactly one solution.
- If `N(A)` contains a nonzero vector, then every consistent system `Ax = b`
  has infinitely many solutions, because you can add scalar multiples of that
  vector to a particular solution.

So null space measures the hidden freedom in the system.

## Common mistakes

<CommonMistake title="The zero vector always belongs to the null space">
Students sometimes think a homogeneous system can have no solution. That is
impossible, because `x = 0` always satisfies `Ax = 0`.
</CommonMistake>

<CommonMistake title="A particular solution is not the whole solution set">
Finding one vector `x_p` with `Ax_p = b` is only the start. You still need to
add the whole null space to describe every solution.
</CommonMistake>

## Quick checks

<QuickCheck prompt="Why does `Ax = 0` always have at least one solution?">
Answer in one sentence.
</QuickCheck>

<RevealSolution title="Answer">
Because the zero vector always satisfies `A0 = 0`.
</RevealSolution>

<QuickCheck prompt="If `N(A) = \{0\}` and `Ax = b` is consistent, how many solutions does it have?">
Use the theorem from this note.
</QuickCheck>

<RevealSolution title="Answer">
Exactly one, because there is no nonzero null-space vector to add to a
particular solution.
</RevealSolution>

<QuickCheck prompt="If `Ax = 0` has a free variable, can the solution set contain only two vectors?">
Answer from the parameter form, not from a guess.
</QuickCheck>

<RevealSolution title="Answer">
No. A free variable can vary through infinitely many scalar values, so it
produces infinitely many solution vectors.
</RevealSolution>

## Exercise

<QuickCheck prompt="Suppose `x_p` solves `Ax = b` and `u, v \in N(A)`. Why do `x_p + u` and `x_p + v` both solve `Ax = b`?">
Write one line using linearity.
</QuickCheck>

<RevealSolution title="Guided solution">
Because `Au = 0` and `Av = 0`, we have

$$
A(x_p + u) = Ax_p + Au = b + 0 = b,
$$

and similarly

$$
A(x_p + v) = Ax_p + Av = b + 0 = b.
$$

So adding any null-space vector to a particular solution keeps you inside the
solution set.
</RevealSolution>
