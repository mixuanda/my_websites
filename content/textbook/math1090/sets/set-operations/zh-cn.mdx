## 什么是集合

<Definition title="集合">
集合是事物的集合。

若 `x` 是集合 `A` 的元素，我们写作 `x ∈ A`；若不是，则写作 `x ∉ A`。
</Definition>

两个集合相等，表示它们的元素完全一样。

## 主要运算

| 运算 | 符号 | 读法 |
| --- | --- | --- |
| 并集 | `A ∪ B` | 属于 `A` 或 `B` 的元素 |
| 交集 | `A ∩ B` | 同时属于 `A` 和 `B` 的元素 |
| 差集 | `A \ B` | 属于 `A` 但不属于 `B` 的元素 |
| 补集 | `A^c` | 在选定的全集中，位于 `A` 外的元素 |

## 用一个具体例子看运算

<WorkedExample title="追踪元素如何移动">
设 `A = {1, 2, 4}`，`B = {2, 3, 4}`。

那么：

- `A ∪ B = {1, 2, 3, 4}`
- `A ∩ B = {2, 4}`
- `A \ B = {1}`

如果全集是 `E = {1, 2, 3, 4, 5}`，则 `A^c = {3, 5}`。

<RevealSolution title="快速检查德摩根律">
某个元素不在 `A ∪ B` 里，正好等于它同时不在 `A` 里，也不在 `B` 里。这就是 `(A ∪ B)^c = A^c ∩ B^c`。
</RevealSolution>
</WorkedExample>

## 另外两个你会见到的构造

本地材料还会介绍两个常用构造：

- `A × B`，笛卡儿积，是有序对 `(a, b)` 的集合。
- `P(A)`，幂集，是 `A` 的所有子集所组成的集合。

这两个符号值得及早认识。

## 常见错误

<CommonMistake title="不要把补集和差集混为一谈">
`A^c` 要先有全集。`A \ B` 则是另一个集合运算。两者不是同一回事。
</CommonMistake>
